# Airline-Flight-Data-Analysis-Report
This project performs Exploratory Data Analysis (EDA) on an airline flights dataset to uncover key patterns, pricing trends, and factors affecting ticket prices.  It includes data cleaning, feature understanding, and visualization using Python libraries like Pandas, Matplotlib, and Seaborn.
✈️ Airline Flights Data Analysis
📌 Overview

This project focuses on Exploratory Data Analysis (EDA) of airline flight data to identify patterns in ticket pricing, airline performance, and travel trends.

The goal is simple: convert raw data into actionable insights using Python.

🎯 Objectives

Analyze how ticket prices vary across different airlines

Understand the effect of stops, class, and booking timing on pricing

Identify trends in flight routes and travel patterns

Detect outliers and anomalies in the dataset

📊 Key Analysis

Distribution of airlines and flight frequency

Source → Destination route analysis

Price variation by:

Airline

Number of stops

Travel class (Economy vs Business)

Relationship between:

Price vs Days Left for Departure

Outlier detection using boxplots

Trend visualization using scatter and line plots

🛠️ Tech Stack

Python

Pandas – data manipulation

NumPy – numerical operations

Matplotlib – basic visualization

Seaborn – advanced visualization

🔧 Steps Performed

Data Loading and Inspection

📈 Key Insights

Ticket prices increase as departure date approaches

Business class fares are significantly higher than economy

Flights with fewer stops tend to be more expensive

Certain airlines consistently charge higher fares

Price distribution contains outliers indicating premium bookings
